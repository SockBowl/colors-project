# colors-project

a web app for creating new color palettes.

## Description

Users can take a look at existing color palettes or add their own color palette.
Data is currently not persistent.

### Installing

- clone repository

### Executing program

- In terminal run

```
npm i
```

- Then run

```
npm start
```
